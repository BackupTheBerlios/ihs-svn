Chapter 1. Introduction

Following chapter includes short introduction to this project. It includes lisf of requirements and other constraints. There's also a glossary.

This document is a detailed documentation related to this project. It includes: introduction, design considerations, system architecture, policies and guidelines, detailed design, glossary, bibliography and administrator's guide.

Document is for anyone that wants to get familiar with the system. Some parts require IT knowledge. Target groups are administrators and developers.

Full name of the project is Incredible Hosting System, acronym is IHS. Usually acronym will be used. Main part of IHS is administration panel that allows administrators to manage mail system. It was my master thesis.

To get familiar with this project, made decisions and to learn about some products it's worth checking my engineer thesis - Free Mail System - NIE-2003. IHS is based on experience gained during implementation of previous thesis, it's another step to achieve integrated management framework for hosting systems.
